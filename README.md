# Tesla Lightshow Studio

A modern desktop app to create Tesla-compatible light shows.

## Features

- Multi-car support (MVP: 10 vehicles)
- 2D & 3D preview
- Timeline-based animation
- Tesla-compatible export (.fseq)

## Stack

- React + Zustand
- Tauri (Rust)
- Three.js

## Getting Started

```bash
npm install
npm run dev
```


In apps/tauri
```bash
npm install
npm run dev
```
