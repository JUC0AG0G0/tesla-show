# Specs

Bienvenue dans les spécifications du Tesla Light Show Editor.

Ce système permet de synchroniser et piloter des shows lumineux sur des flottes de véhicules en temps réel.

Tout ce qui est ici définit les règles du système, les protocoles et les limites physiques du monde réel (oui, les voitures ont encore des batteries).
